#  Redux-sport

### Описание проекта:
В данном проекте создано приложение для учета спортивных тренировок. 
Реализована возможность добавить, удалить, отфильтровать, отсортировать тренировки.
Тренировки (tasks) хранятся в Redux сторе.

### Публикация:
https://alsu-dxdy.github.io/redux-sport/

### Стек проекта:
- React
- Redux
- JS

### Планы по развитию проекта:
Добавить:
возможность редактирования тренировки,
сохранение данных в localstorage,
Redux Form.

### Чтобы начать работу с проектом:

- Клонировать репозиторий:

```
git clone https://github.com/alsu-dxdy/redux-sport
```

- Установить зависимости: в папке проекта в терминале ввести:

```
npm install
```

- Запустить проект: в папке проекта в терминале ввести:

```
npm start
```
- Enjoy!
